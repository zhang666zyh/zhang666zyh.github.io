    void main(){

  print(i);
}

  • 第一种写法只是因为Dart默认main函数返回值为空

  • 还有另一种写法

  • 变量类型 变量名 = value

    1
2
3
4

    void main(){
  String str = 'hello';
  print(str);
}

  • dart是一种强类型语言,你不可以给一个指定类型的变量赋上其它类型的值

  • 没有指定的只要有初值也不行,因为dart会自动推导

assert函数打印

  • assert函数是一个特殊的函数,它接收判断条件,如果判断结果为false则中断当前执行条件

  • assert语句只在开发环境中是有效的,在生产环境中是无效的

  • Flutter中的assert只在debug模式下有效

    1
2
3
4

    void main(){
  assert(1 > 1);
  print({'a':1,'b':2});
}

  • 例如你这样,它就会报错且在assert函数的位置终止运行

Step07.常量

  • 常量分为const和final两种

    1
2
3
4
5
6
7

    void main(){
  const i = 1;
  print(i);
  
  final j = 0;
  print(j);
}

  • const必须在定义的同时就赋予初值(否则会报错)

  • 而final则很灵活

  • 当然也可以这样写:

     1

    void main(){
  final String str = 'hello';
  print(str);
      
  final nowTime = new DateTime.now();
  print(nowTime);
    
  const List<int> lists = [1,2,3,4,5];
  print(lists);
}

  • 这样就相当于定义了常量的类型

void main(){
  //定义字符串的两种方式
  //1.直接通过var声明
  var str1 = 'hello';
  print(str1);
  //2.通过对象的方式声明
  String str2 = 'Hello';
  print(str2);

  //Dart中的字符串不区分单引号和双引号

  //多行字符串
  String str3 = '''
                Java
                Python
                GoLang
                Dart
                Kotlin
                ''';
  print(str3);

  //字符串的拼接
  //1.通过传统的+拼接
  var s1 = 'hello';
  var s2 = 'world';
  print(s1 + s2);

  //2.通过${}传值拼接
  var ss = 'yu';
  print('zhang${ss}hang');

  //字符串值可以比较是否相同,不可以比较大小
  var a = 'hello';
  var b = 'world';
  print(a == b);

  //字符串的长度与下标
  String username = 'zhangyuhang';
  print(username.length);
  print(username[2]);

  //切割字符串,变成集合
  String password = '123456';
  print(password.split(''));

  //检查是否存在某个字符串
  String user = "12@32gjs[";
  print(user.contains('@'));

  //检测是否以某个字符串开头
  String user2 = '123456';
  print(user2.startsWith('12'));
  //endsWith

  //检测是否存在某个字符串,返回下标,没有返回-1
  //从左往右第一个
  String ooo = 'hellosssooossjdos';
  print(ooo.indexOf('o'));
  //从左往右最后一个
  print(ooo.lastIndexOf('o'));

  //截取字符串
  String bySplitString = 'helloDart';
  //包含起始位置,不包含结束位置,如果不写结束位置默认到最后
  print(bySplitString.substring(2));

  //转换大小写
  String apple = 'aBcD';
  print(apple.toLowerCase);
  print(apple.toUpperCase);

  //去掉首尾空格
  String bbb = '   123  ';
  print(bbb.trim());

  //正则与替换
  var pattern = new RegExp('a');
  String byMatch = 'aaaba';
  byMatch.replaceAll(pattern, byMatch);
  print(byMatch);


  //检测字符串中是否存在字符串
  String find = 'a';
  String tests = 'a123';
}
