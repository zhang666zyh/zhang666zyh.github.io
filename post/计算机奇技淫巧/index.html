卸载安装的多个python

假如你要卸载的是python3.7

1、删除Python3.7框架

1
2

$ ls /Library/Frameworks/Python.framework/Versions/3.7
$ sudo rm -rf /Library/Frameworks/Python.framework/Versions/3.7

2、删除Python3.7应用目录

1
2

$ cd /Applications
$ sudo rm -rf Python\ 3.7/ #Python 3.7存在空格

查看launchpad中python3的IDLE就被删除了

3、删除/usr/local/bin 目录下指向的Python3.7 的连接

1
2
3

$ cd /usr/local/bin/ 
$ ls -l /usr/local/bin    # 查看所有指向已删除的/Library/Frameworks/Python.framework/Versions/3.7的链接文件
$ rm Python3.7*    # 相关的文件和链接 #Python3.7相关的文件和链接需要你自行确认

  • 下载AI配音专家,支持mac系统和Windows系统
  • 进入阿里云官网,点击右上角的头像,进入账户中心
  • 点击右上角头像,Assets key管理
  • 点击弹出框中的继续使用AssetsKey管理
  • 左上角点击“创建AccessKey”
  • 创建成功后保存好AccessKey ID和AccessKey Secret
  • 打开申请AppKey的界面： https://nls-portal.console.aliyun.com/applist
  • 接着点击右上角“创建项目”，输入项目名称，项目类型默认就行（包含语言识别合成都可以），然后点击确定
  • 创建完成之后就能看到AppKey
  • 最后，就是将上面获取到的三个参数：AccessKey ID、AccessKey Secret、AppKey填入软件内，然后点击保存按钮。
