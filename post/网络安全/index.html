Step02.TCP/IP

  • TCP/IP指传输控制协议/因特网互联协议（Transmission Control Protocol / Internet Protocol），又名网络通讯协议。

TCP/IP 是Internet最基本的协议、Internet国际互联网络的基础，由网络层的IP协议和传输层的TCP协议组成，是供连接因特网的计算机进行通信的通信协议。

TCP/IP 定义了电子设备（比如计算机）如何连入因特网，以及数据如何在它们之间传输的标准。

TCP/IP协议从字面上看，有人可能会认为 TCP/IP 是指 TCP 与 IP 两种协议。虽然实际中的确有这两种协议，但是在很多情况下，它泛指 IP、ICMP、TCP、UDP、TELNET、FTP、HTTP 等协议群，所以有时也称 TCP/IP 为 网际协议族。

  • 其实这些不同的协议都是进行网络通信的不同方式或过程,而且在计算机网络四层或七层模型中的第几层而已

    例如UDP和TCP的区别,HTTP是BS架构的超文本传输协议用于浏览器网站的访问、FTP是文件传输协议等

Step03.IP和端口

  • TCP/IP:网络通信的基础

IP

  • IP是计算机在互联网中的唯一标识

  • 用来在互联网中寻找电脑

  • 1
2
3
4
5

    例如```张三```的电脑```ip```是``123.456.789```
​```李四```的```ip```是```111.111.111.111```
张三想访问李四,就需要张三和李四互相知道对方的```ip```地址
而普遍的网络访问一般都是```域名访问```的
其实用到了```DNS```技术将```域名```解析为```ip```地址

内网IP(局域网IP)和外网IP(互联网IP)

  • 内网ip

    指的是路由以内的IP,可以直接连接互联网,但外网无法直接访问内网,需要交换机或者路由器这种进行端口映射

  • 公网IP

    指的是互联网中的IP,可以直接和互联网互通,不需要进行端口映射,日常像摄像头、远程连接服务器以及主机游戏之间互联等

ip地址就像家庭地址一样,公网IP就是小区地址,内网IP就是指定人家的门牌号,你从家里走出小区(内网访问外网)没人管你,但是你进入其他人家(外网访问内网)就需要指定的安全验证

  • 我试着通过mac上的终端去ping安装在mac虚拟机中的Windows10的IP地址,会被Windows防火墙给墙掉

端口

端口就是计算机中应用程序的唯一 -- 标识

  • ip和端口的关系就是

    ip是一个电脑,当访问这台电脑时候,访问某个应用软件就需要访问对应的端口

    例如使用QQ给别人发送消息,QQ就有一个端口

    通常电脑邮箱、浏览器都有通用端口

端口的分类

  • 端口号从0~65535

  • 公认端口/带名端口/保留端口

    1
2

    端口号从`0~1023`,他们紧密的绑定与连接一些特殊的服务,不可再重新定义它的作用对象
例如`FTP`是21端口,`SMTP`是25端口,`80`端口是HTTP服务,`SSH`默认是22等

  • 注册端口

    1
2
3

    注册端口范围从`1024~49151`,它们松散的绑定一些服务,但也有许多服务可以使用这些端口
这些端口有些既被绑定了一些固定服务,也可以应用于其他应用程序
只要运行的程序向系统提出访问网络的申请,那么系统就会从这些端口中`(从1024开始)`分配一个供该程序使用(例如Redis集群就需要使用多个端口),在应用程序关闭后,端口就会被释放,其他程序可以进行使用

  • 动态/私有端口

    1

    私有端口范围是`49152~65535`,理论上不应将这些端口分配给服务,但实际上一些较为特殊的程序,特别是一些木马程序就喜欢使用这些端口,因为这些端口通常不被人们注意,便于隐藏

Step06.用户和组

  • 一个电脑可以有多个系统,一个系统可以有多个账户

  • 不同账户有不同的权限

  • 每个账户有自己唯一的SID

    用户SID:S-1-5-21-310440588-250036847-580389505-500

    系统SID:S-1-5-21-310440588-250036847-580389505

    • 用户SID比系统SID多一个500是因为,同一系统中所有账号最前面都是相同的,都是系统SID,后面加上不同数字代表不同权限的账户
    • 系统SID是电脑的标识
    • 用户SID是在系统SID的基础上加上不同的数字用来代表用户ID

    用户UID:500

    Windows系统管理员账户administrator的UID是500

    普通账户的UID是从1000开始

  • Windows系统密码

    默认存储位置在C:\Windows\System32\config\SAM中

    这种存储方式是Windows的加密算法,这种加密方式是不可逆算法(只允许单向加密不允许解密也无法解密,将密码字符串转换为哈希值-无论多长、什么形式最终都会转换为4个字节的哈希值,用于验证密码完整性)

    那Windows系统第二次输入密码时是怎样验证密码有效性的呢

    将每一次输入的密码转换为哈希值,然后和SAM中的哈希值进行比对,如果相同证明密码正确

    破解的方法一般有两种

    • 撞库

      一些人将大量的现有哈希值和对应的密码存储在库中,然后通过在库中查找的方式进行筛选

    • 暴力破解

      就是有限制条件或无限制条件的穷举匹配

      例如在生日、身份证号、姓名、居住地址等字段之间排列组合进行暴力穷举密码

    Windows系统服务器中密码默认有效时长为42天,为了避免别人暴力破解自己的服务器,服务器每隔42天会强制用户进行密码更改

Step08.文件系统

NTFS权限

文件系统即在外部存储设备中存储文件的方法

  • 文件系统

  • Windows

    NTFS

    FAT

  • Linux

    EXT

  • 这是一块磁盘/存储空间
  • 存储空间是由多个小格子组成的,称为存储单元
  • 例如这块大的存储空间是4GB,那么一个单元格的空间可能只有几kb(这个可以自己设置)
  • 将磁盘格式化就是将这块存储空间中所有内容全部清空,重新划分格子,这个过程就叫做格式化
  • NTFS、FAT以及EXT等不同分区格式指的就是划分格子的不同
  • 比如你要在这个磁盘存储一个4GB的电影,每个格子只有几百kb,就需要将电影文件拆分成不同的片段塞到格子中
  • 但是需要注意的是,比如创建了一个txt文件,而没有写入任何东西,它是不占用硬盘空间的,因为在这块磁盘的左侧有一条文件目录,用来存储文件索引

  • 在将磁盘格式化时,可以指定文件系统、分配单元大小等
