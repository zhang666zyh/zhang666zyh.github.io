Step06.C++的三字符组

三字符组就是用于表示另一个字符的三个字符序列，又称为三字符序列。三字符序列总是以两个问号开头。

三字符序列不太常见，但 C++ 标准允许把某些字符指定为三字符序列。以前为了表示键盘上没有的字符，这是必不可少的一种方法。

三字符序列可以出现在任何地方，包括字符串、字符序列、注释和预处理指令。

如果#if后面的值为false,则不会执行#if里面的代码

所以在写项目的时候可以使用#if代替注释,在测试代码的时候将#if后面的值改成true,然后上线的时候将其改为false,这样在项目上线的时候测试代码不会执行

#include <iostream>


using namespace std;

// 对于C++中的全局变量
    // 在其他函数中可以定义重名变量
    // 在函数中调用时它会优先考虑局部变量而无法直接调用全局变量
    // 调用全局变量可以使用    ::全局变量   的方式
int a = 20;


// C++中的命名空间,为了防止一个文件中变量重名定义
    // 要我说namespace就是定义了一个类,然后将全局变量作为它的对象
    // 然后通过    类.对象   这种调用
namespace B1{
    int b = 20;
}

namespace B2{
    int b = 30;
}

// C++的无名命名空间等价于C语言中的static,指定变量或函数只能在本文件中使用
namespace{
    int c = 40;
    void fun(){
        cout << "I do not have namespace" << endl;
    }
}

// 给命名空间起别名,其实就是给这个命名空间起了两个名字,在当前作用域范围内用哪个都行
namespace D{
    int d = 50;
}

namespace E{
    int e = 10;
}

namespace F{
    int f = 10;

    void func(){
        cout << "I am func()" << endl;
    }
}

namespace G{
    void func(){
        cout << "I am func" << endl;
    }

    void func(int a){
        cout << "I am func too" << endl;
    }

    void func(int a, int b){
        cout << "I am func also" << endl;
    }
}

int main(){
    int a = 10;
    cout << "a = " << a << endl;
    cout << "a = " << ::a << endl;

    cout << "b = " << B1::b << endl;
    cout << "b = " << B2::b << endl; 

    cout << "c = " << c << endl;
    fun();

    namespace DD = D;
    cout << DD::d << endl;
    cout << D::d << endl;

    int e1 = 20;
    // 使用命名空间,就可以直接在本空间作用域中使用命名空间的变量而不需要使用 命名空间::变量名 这种方法
    using namespace E;
    // 先在命名空间中查找,如果没有则再到其他地方查找
    cout << "e = " << e << endl;
    
    // 可以直接通过这种使用指定命名空间中的变量的形式在本函数中导入这个变量,就可以直接使用
    // 不需要再using namespace了
    // 一旦使用这种方法就不能在本函数中再次定义同名变量
    using F::f;
    cout << f << endl;
    // 而函数必须使用空间名::函数名这种调用,不能像变量一样一旦通过using 命名空间::变量 就可以不声明任何东西了
    F::func();

    // 函数的重载
    // 切记,不能using name func
    // 可以直接using 命名空间::函数名
    // 这样就可以调用所有名为func的函数了
    using G::func;
    func();
    func(1);
    func(2,3);

    int h;
    // std就是gcc自带的命名空间
    // cin和cout都是其中的关键字
    // 在全局中导入过这个命名空间后,可以直接使用其中的关键字 方法 变量等
    // 也可以通过命名空间::关键字的方式使用
    cin >> h;
    cout << h << endl;
    std::cout << h << endl;

    return 0;
}
  • 在其他函数中使用命名空间:using namespace 空间名
  • 使用命名空间中的某个方法或变量:using 方法名/变量名

#include <iostream>


using namespace std;

struct student{
    int age;
    char name[20];

    // C语言中结构体只能声明成员变量,不能声明成员函数
    // 而C++既可以声明成员变量,又可以声明成员方法
    void func(){
        cout << "我是结构体中的function" << endl;
    }
};

int main(){
    // 在C语言中声明结构体变量需要在前面加上typedef,而C++可以省略
    student stu1 = {19,"zhang"};

    cout << stu1.age << "---" << stu1.name << endl;
    stu1.func();

    return 0;
}

Step12.使用const代替define

  • 为什么使用const替换define

    1

    #define MAXSIZE 100

  • 因为define在预处理-运行前就将代码中所有的MAXSIZE替换成了100

  • 所以这之后报错,即使是MAXSIZE出了问题,编译器也只会认定是100报的错

  • 这样对排除错误工作是很不友好的

  • 而const可进行编译器安全检查,有类型加快了排错工作
