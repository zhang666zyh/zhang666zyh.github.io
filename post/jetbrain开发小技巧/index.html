Step1.快捷键

  • Alt + Insert

    Java类中"生成”

    不过项目开发中一般用lombok

  • Alt + 鼠标左键垂直滑动

    多行同时编辑

  • Ctrl + Alt + S

    打开设置

  • Alt + Enter

    自动导包

    自动补全声明变量

  • Shift + Shift

    全局搜索

    Ctrl + F只能当前文件搜索

  • Tab

    生成对应缩写的代码

    例如写HTML的时候写div.box,点击Tab键,就会自动生成<div class="box"><div>

    w100px,就会生成width:100px

  • Shift + Enter

    光标换行,不影响当前行的文字

  • Ctrl + Enter

    运行选中的SQL语句

  • Ctrl + F4

    关闭当前文件窗口

Step11.webstorm自动将px转为rem

  • setting -> plugins
  • 搜索px2rem
  • 下载
  • setting
  • 左侧就出现了px to rem选项
  • 更改root fontsize
  • 这个值 / 100,假设等于n
  • 那就是m(px) * n = m * n(rem)
