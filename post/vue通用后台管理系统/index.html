Step06.点击header左侧的按钮,控制左侧侧边栏的伸缩

  • elementui这个侧边栏,是通过@close="handleClose" :collapse="isCollapse"属性控制是否伸缩的

  • 因为你是要在header中点击,在aside中触发缩放

  • 这就涉及到了父子组件之间的事件传递

  • vuex比$emit、props好用太多

  • 在项目根目录创建store目录

  • 在store目录创建index.js

    export default {
    // state的状态
    state: {
        isCollapse: false
    },
  
    // 改变state的方法
    mutations: {
        collapseMenu(state) {
            state.isCollapse = !state.isCollapse;
        }
    }
}

    • state中,声明要操作的数据

    • mutations, 声明一个方法,这个方法可以操作state的数据

      因为你无法直接操作state,只能通过vuex的mutations方法

        },
        // 面包屑路由跳转
        selectMenu(state, value) {
            if (value.name !== 'home') { // 如果当前组件中的value.name不是首页
                state.currentMenu = value; // 那就让当前的currentMenu为你点击的这个
                const result = state.tabsList.findIndex(item => item.name === value.name) // 将点击的这个的数据,在tabsList(面包屑数组)里查找
                // 如果查不到,就将你点击的这个推送到数组中
                if (result === -1) {
                    state.tabsList.push(value);
                }
                // 到时候遍历这个数组一显示就可以了
            }
        },
