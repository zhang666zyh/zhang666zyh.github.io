Step01.开发环境配置

开发工具介绍

  • 进入uniapp官网,下载开发工具Hbuilder
  • 点击DOWNLOAD,下载对应系统的App开发版,因为它集成了项目所需要的环境,不需要二次导入
  • 我使用windows,下载是一个zip
  • 解压zip,运行exe
  • 按F1打开帮助文档

创建并运行第一个项目

  • 打开编译器,文件->新建->项目
  • uniapp,默认模板,设置好项目名称和位置就好了,其他的不用管
  • 打开项目,菜单栏运行,运行到浏览器

### 封装导航栏为组件并在主页中使用

+ 在项目中,这只是一个导航栏,如果把代码都写在index.vue里面
+ 代码将变得极其混乱
+ 在```导入uni-nav-bar时生成的目录```新建目录```navbar```
+ 在```nav-bar```下新建```navbar.vue```
+ 将刚才在```index.vue```中写的代码全部复制到```navbar.vue```中
+ 将```index.vue```中代码全删除,重建vue模板
+ 只需要通过vue```导入并使用组件```的三部曲
+ import:导入资源
+ component:将导入的资源在当前文件中声明为组件
+ 以标签的形式使用组件
