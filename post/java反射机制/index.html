Step1.什么是反射机制

JAVA反射机制是在运行状态中 对于任意一个类 都能够知道这个类的所有属性和方法

对于任意一个对象 都能够调用它的任意一个方法和属性

这种动态获取的信息以及动态调用对象的方法的功能称为java语言的反射机制

Step2.获取任意类的字节码对象的几种方式

Class类:

Class类的实例对象表示正在运行的Java应用程序中的类和接口

也就是jvm中有很多的实例 每个类都有唯一的字节码对象(Class对象) 通过唯一的字节码对象就可以反射任意一个类中的构造，成员变量和方法

Class类没有公共构造方法 Class对象是在加载类时由Java 虚拟机自动构造的 我们不需要创建 JVM已经帮我们创建了

Class对象用于提供类本身的信息 比如有几种构造方法 有多少属性 有哪些普通方法

package com.lezijie.note;

import java.lang.reflect.Method;

class Test{
    public static void function1(){

    }

    public static void function2(){

    }

    public static void function3(){

    }

    public Integer add(Integer param1,Integer param2){
        return param1 + param2;
    }
}

public class TestAll {

    public static void main(String[] args) {
        try {
            //生成新对象
            Class classType = Test.class;
            Object object = classType.newInstance();

            //通过Class对象获取方法
            Method addMethod = classType.getMethod("add",new Class[]{Integer.class,Integer.class});
            
            //通过生成的新对象来调用这个方法,返回值仍然是一个对象
            Object result = addMethod.invoke(object,new Object[]{1,2});
            System.out.println(result);

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

package com.lezijie.note;

import java.lang.reflect.Constructor;
import java.lang.reflect.Method;

public class TestAll {

    public static void main(String[] args) {
        try {
            Class classType2 = Test.class;
            Constructor constructor = classType2.getConstructor(new Class[]{});
            Object object2 = constructor.newInstance(new Object[]{});
            System.out.println(object2);
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
