Step1.创建第一个Kotlin项目

  • 打开idea,创建Java项目,在其他库和框架中勾选Kotlin

  • 下一步

  • 给项目命名,然后Finish

  • 在项目下新建一个子模块,命名为Kotlin01

  • 创建这个子模块时候选择JVM/Kotlin就行了

  • 点击确定

  • 在子模块的src目录下新建一个kotlin文件,命名为Hello.kt

  • 编写我们的第一个Kotlin代码

    • 变量命名规则:
      • var(说明这是一个变量) 变量名:变量类型 = 值

  • KotLin中的基本数据类型就是Java中的基本数据类型对应的类(开头字母大写)

Step2. 变量

普通变量(var 变量名:变量类型=value)

fun main(){
    var i:Int = 10
    i += 1
    print(i)

  • 对于i,是可读可写的普通变量
  • 普通变量使用var定义

fun main(){
    val i:Int = 1
    print(i)
}
  • 使用val定义只读变量,只读变量不可修改

    * @Contact  :   zhangzyh666@163.com
    * @MyWeb    :   https://zhang666zyh.github.io

fun main(){
    var age:Int = 8
    if(age in 0..3){
        print("婴儿")
    }else if(age in 3..10){
        print("幼儿")
    }else{
        print("少儿")
    }

    if(age !in 0..7){
        print("测试")
    }
}

// @CharacterEncoding   :   UTF-8
/*
    * @File     :   `when`
    * @Time     :   2021/10/11
    * @Author   :   张宇航
    * @Contact  :   zhangzyh666@163.com
    * @MyWeb    :   https://zhang666zyh.github.io
*/

fun main(){
    var school = "小学"
    var level = when(school){
        "学前班" -> "幼儿"
        "小学" -> "少儿"
        "中学" -> "青年"
        else->{
            println("未知")
        }
    }
    print(level)
}
